# 系统工具领域知识

## time_skill
获取当前日期和时间。直接调用即可，无需参数。

## run_python
在沙箱进程中执行 Python 代码。代码在独立子进程中运行，与主进程隔离。

**沙箱特性：**
- 独立进程执行，不会影响主进程
- 默认超时 30 秒，可通过 `timeout` 参数调整（最大 120 秒）
- 内存限制 512MB
- 自动清理临时文件
- 环境变量中的 API_KEY、SECRET、TOKEN 等敏感变量会被移除

**参数：**
- `code`：要执行的 Python 代码（支持多行）
- `timeout`：执行超时秒数（默认 30，最大 120）

**使用场景：**
- 数学计算、数据处理
- 文本转换、格式处理
- 简单的文件操作（注意：工作目录为系统临时目录）

**注意事项：**
- 代码无法访问项目目录（工作目录为临时目录）
- 无法导入需要额外安装的第三方包（除非已在环境中安装）
- 超时代码会被强制终止
